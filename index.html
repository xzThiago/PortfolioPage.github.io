Projeto em Destaque

Data Warehouse & Analytics

Este projeto demonstra a construção de uma solução completa de Data Warehouse e Analytics, desde a ingestão de dados até a geração de insights estratégicos para o negócio.

A arquitetura segue o padrão Medallion Architecture (Bronze, Silver e Gold), garantindo organização, qualidade dos dados e otimização para análises analíticas.

  • 🏗 Arquitetura: Medallion Architecture (Bronze, Silver, Gold)
  • ⚙️ ETL Pipelines: Ingestão de dados CSV para SQL Server com processos de transformação
  • 🗄 Modelagem: Star Schema com tabelas fato e dimensão
  • 📊 Analytics: Dados preparados para relatórios e geração de insights estratégicos
